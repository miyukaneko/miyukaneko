Exercise 1


This was our first exercise with processing; although I have worked with processing in ART 75, since I took that class two years ago I realized I have forgotten a lot of the basics.

The strand of transparant squares and circles follow the mouse's x-axis and y-axis movement. This includes basic drawing functions, basic color functions, basic variables, operator shortcuts and random commands. The sine wave in the background was an example of an nested for loop statement.

Exercise 2


The second exercise includes toggle logic with "if" statements, local vs global variables, boolean logic, loading images, creating text, and key/mouse events. The text jitters, the square changes colors on key press, and the image grids (eyes) appear on mouse press.

Although we have worked on if statements before this exercise, seeing the if statements activate with the actions of my mouse/keyboard made it easier to understand for me.

Exercise 3


The third exercise includes custom functions, arguments, and parent functions. The individual face parts and hair pieces were custom functions, under the two parent functions to the corresponding twin. The glasses option as well as the mouth options were arguments. pushMatrix and popMatrix was appied on the pupils.

This exercise taught me the importance of writing my code in a organized, legible way, as the code turned out much longer than any of my other projects. It also helped me understand how code is read and executed from top to bottom, thus working in a layer-like way in prioritizing more recent commands over others.

Project 1


The slideshow to the left is a gallery of images I drew with the brushes made in Project 1.

The last assignment was to create algorithmic brushes and drawings; combing all the experience I have earned from the three exercises, I made a drawing tool with 7 different brushes and an erasor. The brush I worked the hardest on is brush 1 (press key 1), where I made the brush become progressively bigger and cycle through colors (rainbow) by using the "++" increments.

ARTIST STATEMENT: Although this was the hardest assignment so far, I think I got the most enjoyment out of it. This could be used as an simple application like MS Paint, but with a specific aesthetic to it. I used a lot of faded, pastel colors for my simple brush options and all of my brush options are round for a softer results to the images. For brushes 1, 4,5,and 7, I played around with the transparancy levels as it created a fun 3-D like effect to the brushes. I found a lot of difficulties in making brush 1 work the way I wanted it to; I had some issues with it affecting the eraser and it not cycling through the colors correctly, but I'm happy with the end result and I learned a lot from the patches I needed to make. Overall, this was a challenging yet fun project!
