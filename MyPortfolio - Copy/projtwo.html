Project 2


The last assignment was to create a time-based algorithm by applying knowledge of arrays and objects, as well as using the millis command in the algorithms. I made three classes, the mushroom (shape), the flower (shape), and spring (algorithm). I first applied the millis command in the spring algorithm; before the springs would be controled by mouse drags and clicks, but I changed the mouse commands to millis commands. By doing this I can recreate the mouse movements automatically. Then I changed the springing shapes to flowers; this was harder than I thought it would be since the rotation, colors, and scale needed to be tweaked accordingly so that they would cooperate with the spring algorithm. I also added the minim audio library so I had audio the spring movements could sync with.

ARTIST STATEMENT: This was a lot more difficult and complex than I intended it to be. Although I had a good understanding of objects and arrays, having this many components in one project and making them all work in the way I invisioned was a challenge. I kept on wanting to add new things; more shapes, more audio syncing, more background changes, etc, etc...which complicated things, but that was what made the project fun. In the future, I would love to try to find a way to sync the movements in a technical way, since this time it was manual timing.
